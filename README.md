# spring-boot-learning
近些年来 spring boot 瞬间风靡java界，而spring boot 为什么这么火的主要原因在与它的设计理念-**约定优于配置（convention over configuration）**，
约定优于配置是一种设计范式，正由于这样的设计理念才让 spring boot 能够开箱即用，能够在开发过程中减少了开发人员需要作出决定的数量，同时也减少了大量
繁琐的并且重复性的 XML 的配置。
